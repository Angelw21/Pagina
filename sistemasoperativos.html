WINDOWS

El sistema operativo de Windows fue lanzado al publico en los años 1985 el 20 de noviembre, este sistema operativo consta con varias versiones unas de las mas conocidas y usadas es Windows 7. Windows cuenta con 12 versiones la primera Windows 1.0 y la mas reciente Windows 11

DEBIAN

Debian es una de las distribuciones de Linux más antiguas y estables. Fue creada en 1993 por Ian Murdock y es mantenida por una comunidad de desarrolladores y usuarios. A continuación, se presenta una breve descripción de las principales versiones de Debian debian1.0 1 96 y la ultima vercion que tienen debian12.0 2022

UBUNTU

Ubuntu es una distribución popular de Linux basada en Debian, patrocinada por Canonical Ltd. Fue lanzada por primera vez en 2004 y se enfoca en la facilidad de uso, la seguridad y la actualización regular,primer version fue ubuntu 4.10 2004 y la ultima version ubuntu 24.04lts

PARROT OS

Parrot OS es una distribución de Linux orientada a pruebas de penetración, análisis forense y seguridad informática el primer fue Parrot OS 3.0 - Lanzado el 29 de diciembre de 2010 y el ultimo Parrot OS 4.11 - Lanzado el 23 de abril de 2021
